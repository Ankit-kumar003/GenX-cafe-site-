
### Windows

```bash
venv\Scripts\activate
```

---

## STEP 6 — Initialize Git

```bash
git init
```

---

## STEP 7 — Add All Files

```bash
git add .
```

---

## STEP 8 — First Commit

```bash
git commit -m "Initial Commit"
```

---

## STEP 9 — Main Branch

```bash
git branch -M main
```

---

## STEP 10 — Connect GitHub Repo

```bash
git remote add origin YOUR_REPO_LINK
```

Example:

```bash
git remote add origin https://github.com/rahul656676/GenX-cafe-site.git
```

---

## STEP 11 — Push Code

```bash
git push -u origin main
```

---

# 🔥 IMPORTANT FIXES

## If "remote already exists"

```bash
git remote remove origin
```

Then re-add origin.

---

## If "403 Permission Denied"

Wrong GitHub account login.

Fix:

1. Open:

   ```txt
   Credential Manager
   ```

2. Remove GitHub credentials

3. Push again

---

## If GitHub blocks API Key

Remove keys from code.

Use:

```python
os.environ.get("GROQ_API_KEY")
```

Store keys inside `.env`
